Pokemon team builds

React TypeScript Tailwind NextJs Postgresql

This is a team builder built with typescript, nextJs, and react query. Users can log in to save data, view teams they have built, track their stats, view others teams. Extra features allow users to favorite pokemon and teams, as well as pre select data they would like when building pokemon.

Movie Mania

React TypeScript Tailwind Firebase

Created using React, react hooks, tailwind, typescript, and firebase as a backend, Movie Mania utilizes the TMDB api to showcase thousand of movies, allowing users to favorite movies they love, review movies they hate, and explore to find their next favorite movie.

Bug Tracker

React Typescript BEM Figma

Created using React, typescript, BEM naming convention and using a figma design doc for the design, This bug tracker allows users to report bugs, comment on others suggestions, upvote which bugs need to the most attention and see how close bugs are to being fixed.

Ecommerce Store

React TypeScript SCSS Figma

Created using React, react hooks, typescript, and using a figma design doc for the design, Audiophile allows users to shop for an assortment of audio, add them to their cart, and then go through a checkout process to complete the transaction.

Wordle Clone

React hooks vanilla-CSS

Created using react, with a heavy focus on the javascript and the logical side of the project. This is a clone of the popular game Wordle, users have 6 guesses to guess a random word, and as they guess they are given clues as to what the word can be based on their previous guesses.
